# Diabetes Prediction (Logistic Regression)

This project is a simple machine learning model that predicts whether a person is diabetic based on health-related features.

I built this to understand how Logistic Regression works for classification problems and how probabilities are used for decision-making.

---

## Features used

- Glucose
- Blood Pressure
- BMI
- Age

---

## What this project does

It takes input features and predicts whether a person is diabetic or not using Logistic Regression.

---

## Tech used

- Python
- Pandas
- Scikit-learn

---

## How to run

pip install pandas scikit-learn  
python diabetes_predictor.py  

---

## What I learned

- Difference between Linear and Logistic Regression  
- How sigmoid function converts values into probabilities  
- How classification decisions are made using thresholds  
- Importance of feature weights
